Torys

A fantasy world set on a ring-shaped planet. Created by Arlo Watts for use in role-playing games and other media, this work-in-progress describes locations, characters, and standalone adventures set on Torys.

Zorse

A puzzle game originally created by the New York Times, reproduced here after the original became unavailable. Create puzzles involving mash-up phrases and share them with your friends.

Just Dragons

A minimal role-playing game framework designed to be fun and easy to learn in just a few minutes. It is built alongside Torys and integrates seamlessly with Torys' adventures.
